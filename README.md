## 🔧 Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/Gerome-Elassaad/repo-booster.com.git
   ```
2. Navigate into the project directory:
   ```bash
   cd repo-booster
   ```
3. Install dependencies:
   ```bash
   npm install
   ```
4. Start the development server:
   ```bash
   npm run dev
   ```
5. Open the app in your browser:
   ```
   http://localhost:3000
   ```
